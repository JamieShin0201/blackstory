# 자바자바
  좋은 원두를 먹을 자격이 있다.
  
## 1. 서비스 소개
  커피 원두에 대하여 사용자의 취향에 따라 추천해주는 시스템
 
## 2. 참여자
  유상문, 신종명

## 3. 핵심 기능 정리
  1. 동네 카페에서 판매하는 원두를 사용자의 취향에 따라 추천해주는 시스템
  2. 취향이 없다면? 원두의 객관적인 정보를 제공
  3. 문의 게시판
  4. 추후 챗봇

## 4. 개발환경
  AWS, OpenJDK 11, SpringBoot 2.4.x, Nginx, Spring Data JPA, Docker, Thymeleaf, MariaDB
